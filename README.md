# 🐒 Simulación Física: Disparo al Mono (Shoot the Monkey)

Este proyecto es una herramienta didáctica e interactiva de **Física Aplicada** diseñada para visualizar y comprender de manera práctica el clásico experimento teórico conocido como *"Disparo al Mono"* (Shoot the Monkey). La simulación modela los principios fundamentales del **movimiento parabólico** y la **caída libre** de forma simultánea.

## 💻 Cómo Abrir y Ejecutar el Proyecto

### Opción 1: Ejecución Directa (Recomendada)
1. Descarga o clona este repositorio en tu computadora.
2. Localiza el archivo [index.html] en la carpeta raíz del proyecto.
3. Haz **doble clic** sobre [index.html] para abrirlo automáticamente en tu navegador web preferido (Google Chrome, Mozilla Firefox, Safari, Microsoft Edge, etc.).



## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructuración semántica de la página y contenedor de dibujo `<canvas>`.
* **CSS3:** Diseño responsivo con variables de diseño, modo oscuro y animaciones fluidas.
* **Vanilla JavaScript:** Lógica matemática del cálculo físico paso a paso y renderizado en tiempo real a 60 FPS con `requestAnimationFrame`.
